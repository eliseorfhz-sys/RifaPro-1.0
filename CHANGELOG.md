# Changelog

Todos los cambios relevantes de **RifaPro-1.0** se documentan en este archivo.

Este proyecto sigue de forma aproximada el formato de [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/)
y versionado semántico.

## [Unreleased]

### Added

- Estructura base de publicación para GitHub (`README.md`, `LICENSE`, `.gitignore`).
- Guía de colaboración en `CONTRIBUTING.md`.
- CI con GitHub Actions (`.github/workflows/ci.yml`).
- Plantillas de issues, PR y `CODEOWNERS`.
- Cuatro herramientas de sorteo: nombres, comentarios, ruleta y números aleatorios.
- Certificado de transparencia por sorteo.
- Ruleta animada con componente HTML personalizado.

## [1.0.0] - 2026-06-20

### Added

- Primera versión funcional de la aplicación de sorteos con Streamlit.
